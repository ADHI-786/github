# Github Finder

It is react based web application for searching and viewing the github profile and used Github API.
https://github7.netlify.app/

## Local Setup

```sh
$ git clone https://github.com/adhit7/github-finder-react
```

```sh
$ cd github-finder-react
```

```sh
$ npm install
```

## Run Locally

```sh
$ npm start
```
